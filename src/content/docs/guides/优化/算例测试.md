本模块通过 JavaScript 在结构化网格上**复现经典拓扑优化算例**，目标是直观展示优化方法的迭代过程与最终构型，并通过可交互的演示帮助理解算法行为。

## 模块说明

- **网格**：采用结构化四边形网格，便于在浏览器中用 Canvas 高效绘制单元密度场；
- **求解**：以轻量有限元（或简化模型）求解位移场，计算单元灵敏度；
- **更新**：采用 OC 或 MMA 等准则更新设计变量；
- **可视化**：实时渲染密度演化过程，支持参数调节。
